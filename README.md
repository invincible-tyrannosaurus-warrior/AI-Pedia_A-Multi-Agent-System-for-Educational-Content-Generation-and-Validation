# AI_Pedia_Local

In **coder agent demo**

Workflow: 
1. User Input (including file upload, textual input)
2. convert Input into GPT5-mini processable format. (e.g. convert pdf file into URL)
3. GPT5-mini generate coding guidance
4. feed the coding guidance to 'qwen3-coder-flash'
5. 'qwen3-coder-flash' generate code scripts
6. display output from both Models
